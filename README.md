Antigravity IDE Support for Unity

## Features
- Option to choose Antigravity as an editor
- Automatically generates ```.csproj```

## Installation
via Package Manager (Git URL)
1. Go to Window > Package Manager.
2. Click the + icon > Add package from git URL....
3. Enter the URL of this repository: ```https://github.com/MRSilius/AntigravityUnity.git```

## Usage 
1. Go to Edit > Preferences
2. Choose Antigravity IDE as External Script Editor
